# 隐私政策草稿

## 概述

`f-tab` 是一个用于整理浏览器标签页与书签的新标签页扩展。

## 我们处理的数据

扩展在运行过程中可能处理以下数据：

- 当前浏览器标签页的标题、URL、分组信息和固定状态


## 数据的用途

这些数据仅用于实现扩展功能，包括但不限于：

- 在新标签页中展示和整理当前浏览器标签页
- 将标签页保存为书签并进行分组管理


## 本地存储

扩展会使用浏览器提供的本地存储能力保存用户配置和整理结果。

## 联网行为

扩展不会在未经用户操作的情况下，主动向与功能无关的第三方发送数据。

在以下场景中，扩展会根据用户的主动配置或操作向外部服务发送数据：

- 用户主动执行 GitHub Gist 备份、拉取、推送或自动同步


## 权限说明

扩展申请的权限仅用于实现核心功能：

- `storage`：保存本地状态
- `tabs`：读取和管理当前标签页
- `tabGroups`：读取浏览器标签分组
- `http://*/*` 与 `https://*/*`：打开链接、加载图标、访问用户主动配置的远程服务
